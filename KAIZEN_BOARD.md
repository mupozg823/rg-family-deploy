### K-0002: 하드코딩 색상 제거
- **카테고리**: 유지보수성

- **문제**: 구 핑크 `rgba(196, 30, 127)` 잔존

- **변경 파일**: SeasonSelector.module.css, live/page.module.css, schedules/page.tsx

### K-0003: 호버 효과 표준화
- **카테고리**: 일관성
- **상태**: ✅ 완료 (2026-01-14)
- **문제**: 호버 효과 구현 방식 불일치
- **해결**: 글로벌 `.hover-interactive` 클래스 추가
  ```css
  .hover-interactive:hover {
    border-color: var(--color-pink);
    box-shadow: var(--pink-glow-sm);
    transform: translateY(-2px);
  }
  ```
