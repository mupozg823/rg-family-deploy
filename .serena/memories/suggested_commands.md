# Suggested commands

- Install: `pnpm install` (preferred) or `npm install`
- Dev server: `pnpm dev` / `npm run dev`
- Dev monitor: `pnpm dev:watch` / `pnpm dev:fast`
- Build: `pnpm build`
- Start (prod): `pnpm start`
- Lint: `pnpm lint`
- Format: `pnpm format` / `pnpm format:check`
- Unit tests: `pnpm test` / `pnpm test:run` / `pnpm test:coverage`
- E2E tests: `pnpm test:e2e` (UI: `pnpm test:e2e:ui`, headed: `pnpm test:e2e:headed`)
- DB migrate/seed: `pnpm db:migrate`, `pnpm db:seed`, `pnpm db:test`
- Smoke test: `bash scripts/smoke-test.sh`
