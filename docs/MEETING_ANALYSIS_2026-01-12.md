## 1. 문서 종합 요약

### 1.1 공식 기획안 핵심 요구사항

| 영역 | 요구사항 | 우선순위 |
|------|---------|---------|
| **디자인 컨셉** | "Minimal (절제되고 세련된 고급스러움)" - 가볍거나 유희적 분위기 배제 | 🔴 필수 |
| **메인 배너** | 4~5개 배너, 페이드 자동 전환 | 🔴 필수 |
| **LIVE MEMBERS** | 방송 중 멤버 최상단 배치, 핑크색 펄스 애니메이션 테두리 | 🔴 필수 |
| **공지 피드** | 최신 2건 카드 노출 + 핀 고정 기능 | 🔴 필수 |
| **조직도** | 엑셀부/크루부 토글, 피라미드 형태 UI, 프로필 상세 팝업 | 🔴 필수 |
| **시그리스트** | 유닛별/멤버별 필터, 유튜브 임베드 (Lazy Loading) | 🔴 필수 |
| **랭킹** | 게이지 바 시각화, 전체/시즌별 필터, CSV 업로드 | 🔴 필수 |
| **VIP 페이지** | 1~50위 비공개 페이지, 개인 헌정 페이지 (Secret Page) | 🔴 필수 |
| **캘린더** | 구글 캘린더 스타일, 엑셀부/크루부 색상 구분 | 🟡 중요 |
| **팬더TV 연동** | 라이브 상태 스크래퍼, 공지 자동 파싱 | 🟡 중요 |

## 3. 우선순위별 Action Items

### 🔴 Phase A: 긴급 수정 (1/12 ~ 1/13)

```
1. [UI] 컬러 통일 작업
   - Primary: #fd68ba 기준으로 전체 컴포넌트 검토
   - Variation: #fb37a3, #fe9bd2 (hover/pressed/배지)
   - LIVE 상태: 시안색 #00d4ff 유지

2. [UI] hover 시그니처 컬러 전환
   - 카드 컴포넌트: border/shadow 핑크 전환
   - 버튼: hover 시 primary 컬러 적용
   - 타임라인/시그리스트/조직도 카드 모두 적용

3. [UI] 메인 배너 전체 너비
   - Hero 컴포넌트 양 옆 패딩 제거
   - 이미지 width: 100vw 적용

4. [UI] 텍스트 크기 상향
   - 라이브 멤버 카드: 닉네임/역할 폰트 +2px
   - 시청하기 버튼: 폰트 +2px, 패딩 확대

5. [UI] 조직도 레이아웃 수정
   - "RG FAMILY" 타이틀 크게 (bold, 2xl)
   - "Organization" 텍스트 제거
   - 엑셀부/크루부 토글 위치 조정
```

### 🟡 Phase B: 중요 기능 (1/13 ~ 1/14)

```
6. [기능] 조직도 + 라이브 연동
   - 조직도 프로필 카드에 LIVE 상태 배지 추가
   - 상세 팝업에 "방송 중" 표시 + 바로가기 버튼

7. [UI] 캘린더 핑크 그리드
   - 흰색 배경 + 핑크 그리드 라인
   - 엑셀부/크루부 이벤트 색상 구분

8. [UI] 랭킹 금/은/동 컬러
   - 1위: #ffd700 (금)
   - 2위: #c0c0c0 (은)
   - 3위: #cd7f32 (동)
   - 숫자 표기 단순화 (0 제거)

9. [기능] Secret Page 완성
   - `/ranking/[userId]` 개인 헌정 페이지
   - 방명록, 감사영상, 시그니처, 셀카 섹션
   - 권한 체크 (해당 유저만 접근 가능)
```

### 🟢 Phase C: 다음 릴리스 (1/15 이후)

```
10. [백엔드] 팬더TV 연동 엔진
    - 라이브 상태 스크래퍼 (2~3분 폴링)
    - 공지사항 자동 파싱
    - 수동 오버라이드 기능

11. [기능] 알림 시스템
    - 공지/일정 알림
    - 푸시 알림 지원
```

## 4. 디자인 시스템 업데이트 필요사항

### 4.1 컬러 토큰 정리

```css
/* Primary (대부분의 UI 키 컬러) */
--color-primary: #fd68ba;

/* Variation (제한적 사용: hover/pressed, 배지/태그) */
--color-primary-dark: #fb37a3;
--color-primary-light: #fe9bd2;

/* LIVE 상태 (시안색 유지) */
--live-color: #00d4ff;

/* 랭킹 */
--gold: #ffd700;
--silver: #c0c0c0;
--bronze: #cd7f32;

/* 다크 테마 */
--background: #050505;
--surface: #121212;
--card-bg: #0a0a0a;

/* 라이트 테마 */
--background-light: #ffffff;
--surface-light: #f8f9fa;
--card-bg-light: #ffffff;
```

### 4.2 호버 인터랙션 규칙

```css
/* 카드 호버 */
.card:hover {
  border-color: var(--color-primary);
  box-shadow: 0 0 20px rgba(253, 104, 186, 0.3);
}

/* 버튼 호버 */
.button:hover {
  background-color: var(--color-primary-dark);
}

/* 텍스트 링크 호버 */
a:hover {
  color: var(--color-primary);
}
```
