# 멤버 프로필 시스템 구현

## 개요
RG Family 조직도에 12명의 실제 멤버 정보를 추가하고, 프로필 상세 정보(MBTI, 혈액형, 키, 몸무게, 생일)를 관리할 수 있도록 시스템을 확장했습니다.

## 주요 변경사항

### 타입 확장
- `src/types/organization.ts`: `MemberProfile` 인터페이스에 `weight` 필드 추가

### Mock 데이터 업데이트
- `src/lib/mock/organization.ts`: 12명의 실제 멤버 정보 추가 (모두 Excel Unit)
  - 한백설, 해린, 월아, 채은, 가윤, 설윤, 한세아, 청아, 손밍, 키키, 홍서하, 퀸로니

### UI 컴포넌트 수정
- `src/components/info/MemberDetailModal.tsx`: 몸무게 표시 추가

### Admin 페이지 수정
- `src/app/admin/organization/page.tsx`:
  - `MemberProfile` 인터페이스에 `weight` 필드 추가
  - 몸무게 입력 폼 필드 추가
