# K-003 헌정 페이지 Phase 4 완성

## 개요
헌정 페이지(Tribute Page)에 순위별 테마 커스터마이제이션(Gold/Silver/Bronze)을 추가하고, Supabase 연동을 구현했습니다.

## 주요 변경사항

### 1. 순위별 테마 시스템 추가
| 순위 | 테마 | 컬러 |
|------|------|------|
| 1위 | Gold | `#ffd700` |
| 2위 | Silver | `#c0c0c0` |
| 3위 | Bronze | `#cd7f32` |
| 그 외 | Default | `#fd68ba` (핑크) |

### 2. 수정된 컴포넌트
- `TributeGate.tsx` - rank prop 추가, 테마 클래스 적용
- `TributeGate.module.css` - Gold/Silver/Bronze CSS 추가
- `TributePageHero.tsx` - rank 기반 테마 적용
- `TributePageHero.module.css` - 테마별 스타일 추가

### 3. Supabase 연동 구현
`useTributeData.ts`에서:
- `vip_rewards` 테이블 조인 쿼리
- `profiles`, `seasons` 조인으로 데이터 통합
- HallOfFameHonor 형식으로 변환

```typescript
const { data: rewardsData } = await supabase
  .from('vip_rewards')
  .select(`
    id, profile_id, season_id, rank, personal_message, dedication_video_url,
    profiles:profile_id (id, nickname, avatar_url, total_donation, unit),
    seasons:season_id (id, name)
  `)
  .eq('profile_id', userId)
```

### 4. VIP 라운지 링크 수정
- Before: `/ranking/vip/${user?.id}` (존재하지 않는 라우트)
- After: `/ranking/${user?.id}` (올바른 헌정 페이지 라우트)
