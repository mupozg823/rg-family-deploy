# 조직도 트리 구조 개선

## 개요
회의록 요구사항에 따라 조직도를 단순 리스트에서 트리 다이어그램 구조로 개선했습니다. 대표 2인이 상단에 배치되고 하위 멤버들이 연결선으로 연결되는 구조입니다.

## 주요 변경사항

### 레이아웃 구조 변경
- **이전**: Level 기반 (Level 0 → Level 1 → Level 2)
- **이후**: 트리 구조 (Leaders Row → Connector → Members Grid)

### 새로운 CSS 클래스
| 클래스 | 역할 |
|--------|------|
| `.leadersRow` | 대표 2인 상단 중앙 배치 |
| `.mainConnector` | 연결선 컨테이너 |
| `.mainVerticalLine` | 메인 수직선 |
| `.distributionLine` | 가로 분배선 |
| `.dropLine` | 멤버 방향 수직 연결선 |
| `.membersGrid` | 6열 그리드 멤버 배치 |

### 반응형 대응
```css
1024px → 4열 그리드
768px → 3열 그리드
480px → 2열 그리드 + 연결선 숨김
```

### 테마 지원
- Excel 테마: 골드 연결선
- Crew 테마: 화이트 연결선
- Light 모드: 밝은 골드 연결선

## 결과
- ✅ 빌드 성공
- ✅ 대표 2인 상단 배치
- ✅ 하위 멤버 연결선 구조
- ✅ 모바일 반응형 지원
- ✅ 다크/라이트 모드 지원

## 시각적 개선
```
[린아]  [가애]       ← 대표 2인 상단
    │
    ●─────────────   ← 연결선 구조
    │ │ │ │ │ │
[멤버 6열 그리드]     ← 밀도 있는 레이아웃
```
